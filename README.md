## Descrição do Projeto
Este repositório contém todos os materiais desenvolvidos para o projeto Batalha de Bits, apresentado no Criathon de AOC.

O Batalha de Bits é um jogo de tabuleiro educativo inspirado em War, criado para facilitar o aprendizado de Arquitetura e Organização de Computadores (AOC) de forma divertida, competitiva e interativa.

Os jogadores assumem o papel de engenheiros de hardware e precisam otimizar sua “placa-mãe”, distribuindo capacitores (soldados) nos componentes do sistema. Para avançar, respondem perguntas sobre CPU, RAM, Cache, hierarquia de memória, pipeline, gargalos e muito mais.

O objetivo final é conquistar o melhor computador com base na sua missão, aplicando os conceitos estudados em AOC na prática do jogo.

## Como o Jogo Funciona (Resumo)
Cada jogador recebe:

  - Uma Carta de Objetivo
  - 5 capacitores iniciais

Em cada rodada:
  - Rola-se o dado → define quantos capacitores podem ser alocados
  - Jogador escolhe o componente (CPU, RAM, Cache etc.)
  - Responde pergunta de AOC para validar a jogada

Se o componente estiver lotado:
  - Ocorre uma batalha com outro jogador (quiz + dado)

O vencedor é quem cumpre sua condição de vitória (ex.: dominar CPU + GPU). 

## Objetivo do Jogo
- Ensinar AOC de forma prática, ajudando alunos a compreender:
- Modelo de Von Neumann
- Funcionamento da CPU
- Hierarquia de memória
- Conceitos de gargalo
- Organização de hardware
- Relação entre componentes
